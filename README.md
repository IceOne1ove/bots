## О проекте
VideoFinder Bot — VK‑бот для быстрого поиска видео по пользовательскому запросу.  
Ищет контент внутри ВКонтакте и в сети, показывает подборку с превью и ссылками.

## Назначение
Помочь пользователю найти видео по теме без перехода на другие площадки.  
Подходит для поиска клипов, мемов, обучающих роликов, трейлеров и прочего видеоконтента.

## Основные возможности
- Поиск по текстовому запросу (например, «котики смешные»).
- Выдача списка результатов с названием, длительностью, источником и ссылкой.
- Отправка превью (обложки) к каждому видео.
- Выбор видео из подборки через кнопки или ввод номера.
- Фильтры: по длительности, дате публикации, источнику (ВК / интернет / оба).
- Ограничение выдачи до 10–15 результатов для удобства просмотра.

## Источники поиска
- ВКонтакте: поиск по видеозаписям через API VK.
- Интернет: агрегация из публичных видеохостингов (в рамках правил платформ и доступных API).

## Как пользоваться
1. Отправьте боту текстовый запрос.
2. Получите подборку видео с превью, названиями и кнопками выбора.
3. Нажмите на кнопку нужного видео или напишите его номер.
4. Получите прямую ссылку и краткое описание.

## Команды
- `/start` — приветственное сообщение и краткая справка.
- `/help` — список команд и описание функционала.
- `/settings` — настройка фильтров (источник, длительность, сортировка).
- `/clear` — очистка истории запросов в текущей сессии.

## Интерактивные элементы
- Кнопочное меню для выбора видео и переключения страниц подборки.
- Быстрый фильтр по источнику и длительности (короткие/средние/длинные).
- Подсказки при некорректном запросе или отсутствии результатов.

## Технические особенности
- Интеграция с API ВКонтакте для поиска видеозаписей.
- Использование сторонних API для интернет‑поиска (с учётом ограничений).
- Хранение истории запросов в рамках сессии (без длительного хранения персональных данных).
- Логирование ошибок для быстрого исправления сбоев.
- Адаптация под мобильные устройства: компактный формат сообщений и превью.

## Фильтрация и безопасность
- Исключение контента с явными маркерами 18+ (на уровне запроса и результатов).
- Пропуск приватных видео, недоступных по API.
- Проверка ссылок на работоспособность перед отправкой.

## Целевая аудитория
- Обычные пользователи ВК, которым нужно быстро найти видео.
- Авторы сообществ, подбирающие контент для публикаций.
- Любители мемов, клипов и коротких роликов.

## Преимущества
- Всё в одном окне: поиск, выбор и получение ссылки без переходов.
- Наглядность: превью и краткие данные помогают быстрее выбрать видео.
- Гибкие фильтры под разные сценарии использования.

## Ограничения
- Не все видео могут быть доступны из‑за настроек приватности.
- Некоторые интернет‑источники могут ограничивать выдачу через API.
- Возможны задержки при высокой нагрузке на внешние сервисы.

## Планы по развитию
- Сохранение «избранного» и история просмотренных видео.
- Персональные рекомендации на основе запросов.
- Поддержка голосовых запросов (при наличии технической возможности).
- Расширение списка источников и добавление сортировки по популярности.

## Требования к запуску
- Сервер или хостинг с поддержкой Node.js / Python.
- Токен бота VK и права на доступ к API видео.
- Ключи сторонних API (при использовании внешних источников).
- База данных (SQLite/PostgreSQL) для хранения сессий и логов.

## Установка (кратко)
1. Получите токен бота в VK.
2. Настройте переменные окружения (токен, ключи API, параметры БД).
3. Установите зависимости (`npm install` или `pip install -r requirements.txt`).
4. Запустите бота (`node index.js` или `python main.py`).
5. Проверьте работу через тестовые запросы.
